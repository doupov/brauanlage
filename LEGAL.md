# Právní a licenční poznámky

## Dokumentace

Textová dokumentace vytvořená v tomto repozitáři může být dále upravována a sdílena podle licence zvolené vlastníkem repozitáře.

Doporučená licence pro vlastní dokumentaci:

- Creative Commons Attribution 4.0 International, nebo
- MIT pro skripty a pomocný kód.

Samotné přidání této poznámky licenci automaticky nezavádí. Vlastník repozitáře by měl později přidat odpovídající soubor `LICENSE`.

## Původní program Brauanlage

Podle dochovaných komunitních textů byl program poskytován zdarma pro nekomerční použití domácími sládky. Přesné původní licenční podmínky je ale nutné ověřit v instalačním balíčku, manuálu nebo dialogu programu.

Dokud nebude licence spolehlivě doložena:

- neoznačujte původní program jako open source,
- neupravujte původní copyright,
- nepřisuzujte programu licenci, kterou autor výslovně neuvedl,
- veřejnou redistribuci binárních souborů zvažte odděleně od dokumentace.

## Microsoft komponenty

Následující soubory a produkty jsou majetkem společnosti Microsoft a řídí se jejími licenčními podmínkami:

- Windows XP,
- Microsoft Office,
- Visual Basic 6 Runtime a aktualizace,
- Microsoft ActiveX/OCX komponenty,
- Microsoft Forms 2.0 a `FM20.DLL`.

Repozitář by neměl veřejně obsahovat:

- instalační ISO Microsoft Office,
- produktové klíče,
- neautorizované kopie DLL,
- soubory získané z webů s nejasným původem.

Místo toho používejte oficiální odkazy Microsoftu a lokální zálohu legitimně vlastněných instalačních médií.

## Ostatní komponenty

`InpOut32.dll`, `IOWKIT.dll`, DigiTemp a další soubory mohou mít vlastní licence. Před jejich zveřejněním přidejte:

- název autora,
- původní zdroj,
- původní licenční text,
- verzi,
- hash distribuovaného souboru.
